# Student Grade Tracker

## Project Description
Student Grade Tracker is a Java-based application that helps to manage and track students' grades. It allows users to add student details, calculate grades, and view performance records easily.

## Features
- Add student information
- Enter and manage marks/grades
- Calculate average marks
- Display student performance
- Simple and user-friendly interface

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Java Collections (if used)

## How to Run the Project
1. Clone or download this repository.
2. Open the project in VS Code/Eclipse/IntelliJ IDEA.
3. Compile and run the main Java file.
